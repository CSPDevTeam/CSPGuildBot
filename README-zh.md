## ❓为什么要做该项目
> 市面上有很多的机器人软件，但是都需要外部的支持，繁琐的配置导致很难用，本产品就是为了解决这个问题。

## 💻安装
1. 前往[Release页面](https://github.com/CSPDevTeam/CSPGuildBot/releases)下载最新的版本
2. 安装好[LiteLoader](https://github.com/LiteLDev/LiteLoaderBDS/)
3. 将下载的东西放进plugins文件夹中
4. 按注释配置好config文件
5. 启动BDS
